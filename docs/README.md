# Renzu Documentation Site

This is the static documentation site for Renzu, built with React and Vite.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Screenshots

Place your screenshot images in the `public` folder with the following names:

- `screenshot-1.png`
- `screenshot-2.png`
- `screenshot-3.png`

The site will automatically display these images.

## Deployment

The site is automatically built and deployed to GitHub Pages when a new release is created.
